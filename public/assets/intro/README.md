# Phase1 導入カットシーン用画像

このディレクトリには、Phase1（導入カットシーン）で使用する画像ファイルを配置します。

## 必要な画像ファイル

PDFファイル『狙われた経理部』から以下の8枚の画像を切り出して配置してください：

1. **scene_01.png** - 導入：オフィスの日常
   - PDF P2のイラスト部分

2. **scene_02.png** - イベント：メール受信
   - PDF P3のイラスト部分

3. **scene_03.png** - アクション：開封
   - PDF P4のイラスト部分

4. **scene_04.png** - インシデント：実行
   - PDF P5のイラスト部分

5. **scene_05.png** - リアクション：無視
   - PDF P6のイラスト部分

6. **scene_06.png** - 裏側：マルウェア起動
   - PDF P7のイラスト部分

7. **scene_07.png** - 裏側：C&C通信開始
   - PDF P8のイラスト部分

8. **scene_08.png** - 警告：監視室のアラート
   - PDF P9のイラスト部分

## 画像仕様

- **ファイル形式**: PNG または JPEG
- **推奨サイズ**: 横1920px × 縦1080px（16:9）程度
- **ファイル名**: 上記の通り `scene_01.png` 〜 `scene_08.png`

## 切り出し方法

1. PDFファイルを開く
2. 各ページのイラスト部分をスクリーンショットまたはPDF→画像変換ツールで保存
3. 必要に応じてトリミング
4. このディレクトリに保存

## 注意事項

- 画像が配置されていない場合、代替のプレースホルダー画像が表示されます
- すべての画像を配置後、ブラウザのキャッシュをクリアしてください
