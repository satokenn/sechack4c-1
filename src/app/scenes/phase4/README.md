# フェーズ4（調査・分析）デモ

このディレクトリには、フェーズ4「調査・分析」の学習用デモが含まれます。

ファイル:
- `index.html` - デモを表示する単一の HTML ページ
- `style.css` - スタイル
- `app.js` - ドラッグ＆ドロップの簡易ロジック（ブラウザで動作する純粋な JavaScript）

使い方:
1. このファイルをブラウザで開く（ローカルの静的サーバを使うことを推奨）。
   例: `npx http-server . -c-1 -p 8080` など
2. ツールボックスからアイテムをスキャンエリア / ログ解析エリアにドラッグして挙動を確認してください。

備考:
- 現状は簡易デモ実装です。将来的に React + TypeScript + Vite の構成に組み込むことを想定しています。
